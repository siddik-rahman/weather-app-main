# Frontend Mentor - Weather app  
This is a solution to the weather App challenge from Frontend Mentor. The app allows users to search for any city and view real-time weather data along with forecasts.


![Design preview for the Weather app coding challenge](./preview.jpg)


## 🚀 Features  

- 🔍 Search for any city worldwide
- 🌡️ Display current weather conditions 
- 📅 Get a 7-days forecast
- ⏰ Get hourly weather updates 
- 📱 Responsive design for all devices
--- 
## 🛠️ Built with

- HTML5
- CSS3
- JavaScript (vanilla JS) 
- Openweather API (e.g., OpenWeatherMap)
---
## 📚 What I Learned  

- Working with APIs and fetching data
- Handling asynchronous JavaScript(fetch/async-await) 
- DOM manipulation
- Responsive UI design
